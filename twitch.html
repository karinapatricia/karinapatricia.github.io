Twitch Extension for Customizable Live Game Viewing

Overview

This was project done during the first formal UI design class I've taken at Georgia Tech. I worked as UX designer together with 2 engineers to design a desktop extension for Twitch users to customize their viewing layout while watching live Overwatch games.

During this time, I learned to explore a variety of interactions other than navigating from screen to screen. Using drag and drop interactions and a collapsible player stats tab, we were able to provide a way to seamlessly customize viewing layouts without moving away from the live feed and fulfill our users' needs.

THE PROBLEM

Existing all-access tools that allow custom viewing options end up distracting esports fans from the game instead of providing more insight.

THE SOLUTION

Enable users to seamlessly customize viewing options mid-stream using an all-purpose toolbar.

Drag and Drop to Switch Views

Easily drag and drop any live stream point of view you want by dragging and dropping from the all-purpose toolbar. Users can choose to spectate as third person, through a player's first person point view, the overhead view from the map, or default view from the production team.

Select Layouts

Users can choose to spectate multiple views at the same time by simply toggling the number of layouts under a dropdown menu. Then they can drag and drop views they want into the placeholders while watching the game.

Player Stats

Users can draw conclusions based on real-time updated player statistics in the game. To prevent data from cluttering the screen, users can choose what stats they want to see and hide the stats when they don't need to see it. They can also hover on player icons to get a quick snapshot of their stats.
